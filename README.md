# TEAM_SYNAPSE SQAD
-------------------------------------------------------------------------------------------------------------------------------------------------
● Project Overview

Project Name : Luxe Game Launge

The project is developed using:

● HTML5
● CSS3
● Bootstrap
● React JS
● React Hooks (useState, useEffect)
● Local Storage (optional)
● Vercel for deployment

-------------------------------------------------------------------------------------------------------------------------------------------------
Installation steps : 

1) Download Zip file

2)Extract the zip file

3)open in ide 

4)open terminal add command
      A) npm install
      B) npm run dev
      
5)Open local host link in browser and enjoy the games.

-------------------------------------------------------------------------------------------------------------------------------------------------

Features : 

IT Contains =>  3 games 

1)Tic Tac Toe : Classic two-player grid game with winner and draw detection,also score is displayed so your friend dont cheat .

2)Memory game : A gentle matching puzzle with bright cards and a clean, polished rhythm.

3)Guess the code : Crack a 4-digit code with crisp hints and a clean attempt tracker.

-------------------------------------------------------------------------------------------------------------------------------------------------
Deployment Installation : 

1)open terminal add command
      A) npm run build
      
2)Upload Whole folder in github in new repository

3)Open vercel and sign in with git

4)allow your vercel to acess git repository .

5)publish
